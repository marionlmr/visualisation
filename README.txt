
Application R Shiny - Visualisation de l'indice du bonheur dans le monde
Lou-Anne Guillotel, Floriane Mézirard et Marion Lemer

## Fichiers associés au projet
- visualisation.Rproj qui permet d'ouvrir les fichiers r avec le bon chemin d'accès aux données
Le code R permettant de lancer l'application:
- ui.r
- server.r
- global.r
Les deux fichiers de données :
- world-happiness-report-2021.csv
- world-happiness-report.csv
Les documents explicatifs:
- README.txt
- rendu_visu.html

## Description
Notre application vise à expliquer le bonheur dans le monde. 
Nos différents graphiques peuvent être explorés par le biais d'une interface utilisateur graphique, sans avoir à lire ou écrire de code. 
Les principaux constats à tirer de nos graphiques sont expliqués dans notre document Rmarkdown.


## Mise en route
Vous pouvez explorer notre application de deux manières différentes : 
-Localement en lançant notre projet qui contient les fichiers global.R, ui.R et server.R. Certains packages seront éventuellement à installer.
-En ligne depuis le lien suivant :
https://fmezirard.shinyapps.io/visu_bonheur/
